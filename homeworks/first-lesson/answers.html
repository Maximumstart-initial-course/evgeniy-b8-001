Что такое и зачем нужен HTML?

Это язык разметки, для создания и разметки веб страничек.

Что такое и зачем нужен Git?

Это программа для контроля версий разработок.

Что такое SEO?

Это комплекс мероприятий принятых для поднятия рейтинга(ранга) веб страничек для поисковиков.

В чем отличие Git от GitHub?

Git это программа а GitHub это один из возможных репозитариев для контроля версий.

Что такое тег? Что такое атрибут?

Тег это структырный элемент языка разметки HTMl а атрибут описывает его свойство(характеристику).

Чем отличается язык разметки от языка програмирования? HTML можно назвать языком программирования? Аргументируйте свою позицию.

язык разметки только определяет свойства контента а язык программирования - набор инструкция для ПК. Нет. HTML не выполнит элементарную инстркцию для какой-либо задаци ПК

Зачем нужен атрибут alt у изображений?

Атрибут для описнания элементов тега <img>, а так же для прочтения описания скринридером.
