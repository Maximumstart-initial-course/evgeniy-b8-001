Зачем используются ARIA-роли и ARIA-атрибуты?

ARIA - это стандарт разработанный для людей с ограниченными возможностями. Роли определяют тип элемента и указывают цель, которой он служит. Все атрибуты ARIA используют префикс aria- и делятся на две группы: состояния и свойства. Значения атрибутов-состояний изменяются в результате взаимодействия с пользователем. Значения атрибутов-свойств, как правило, остаются неизменны.

Зачем используются теги `<iframe>`, `<audio>`, `<video>`?

<iframe> это блок в виде специальной прямоугольной области с собственными полосами прокрутки (scrollbars). Это страница внутри страницы. Документы, которые будут лежать внутри будут так же иметь свою собственную размету. Пример использования этого тега - это встраивание видео с, например, ютуба, который дает нам разметку и стилизацию собственного плеера и подгружает его со своих серверов к нам на страницу.Теги <audio> и <video> служат для вставки на веб страницы медиа файлов (аудио, видео).

Какие хорошие и плохие стороны формата JPEG?

За что отвечает свойство display?

Свойство display отвечает за вывод и визуальное отображение элементов на странице. Также с помощью свойства display можно изменить тип генерируемого контейнера.

Чем отличается width: 100% от width: auto?

Когда мы зададим блочному элементу код width:100%, наш элемент (element) растянется на всю ширину родительского блока, а когда задать width:auto, наш блочный элемент поместится внутри родительского элемента, даже если в нем присутствуют margin, padding и border.

Зачем нужно свойство box-sizing?

Свойство box-sizing может принимать одно из двух значений – border-box или content-box. border-box задают значения width/height всего элемента. content-box это значение по умолчанию,свойства width/height обозначают то, что находится внутри padding.

Чем отличаются свойства justify-content и align-items?

Для центрирования элемента по вертикальной оси используется свойство align-items. Для центрирования элемента по горизонтальной оси используется свойство justify-content.

Как заставить флексовые элементы переноситься на новую строку?

Использовать свойство order чтобы перемещать их в нужные места.

Назови два способа выравнивания: 1) блочного элемента по центру; 2) строчного элемента по центру.

1.1 margin: auto; 1.2 display:flex; justify-content: center; align-items:center; 2.1 vertical-align: middle; 2.2 text-align: center;

Чем отличается абсолютное позиционирование от относительного?

Абсолютное позиционирование позволяет выдернуть объект из обычного потока документа и сунуть его в конкретную точку на странице. Относительное позиционирование работает так же, как и абсолютное — заданием объекту его положения координатами сверху, справа, снизу и слева. Разница лишь в отправной точке: в положении точки нулевых координат. Для абсолютного позиционирования ею был край экрана браузера.

Что такое Flexbox и CSS Grid?

flexbox (Flexible Box Layout Module) — модуль макета гибкого контейнера — представляет собой способ компоновки элементов, в основе лежит идея оси. Модуль flexbox позволяет решать следующие задачи:

  • Располагать элементы в одном из четырех направлений: слева направо, справа налево, сверху вниз или снизу вверх.
  • Переопределять порядок отображения элементов.
  • Автоматически определять размеры элементов таким образом, чтобы они вписывались в доступное пространство.
  • Решать проблему с горизонтальным и вертикальным центрированием.
  • Переносить элементы внутри контейнера, не допуская его переполнения.
  • Создавать колонки одинаковой высоты.
  • Создавать прижатый к низу страницы подвал сайта.

модуль CSS Grid Layout система двумерного макета, оптимизированного для дизайна пользовательского интерфейса. Главная идея, лежащая в основе макета сетки, заключается в разделении веб-страницы на столбцы и строки. В образовавшиеся области сетки можно помещать элементы сетки, а управлять их размерами и расположением можно с помощью специальных свойств модуля.

Как строить сетки на флоатах (float)? Для чего должно использоваться свойство float?

Флоатные элементы становятся невидимыми для блочных элементов и видимыми для текста, флоатные элементы видят друг друга. Идущие друг за другом флоаты выстраиваются в ряд, пока им хватает свободного места. Если места не хватает, то они начинают переноситься на следующую строчку. Почти как текст. Как раз эта особенность флоатов и позволила применять их для создания сеток.

Чем отличается Flexbox свойство justify-content от align-items?

Для центрирования элемента по перекрёстной оси используется свойство align-items. Для центрирования элемента по главной оси, используется свойство justify-content.

Зачем используется Flexbox свойство flex-basis?

Свойство flex-basis задает базовый размер флекс элемента по основной оси. Свойство устанавливает начальный основной размер flex-элемента до распределения свободного пространства в соответствии с коэффициентами гибкости. 

Чем отличается flex-grow от flex-shrink?

Свойство определяет коэффициент роста одного flex-элемента относительно других flex-элементов в flex-контейнере при распределении положительного свободного пространства. Свойство указывает коэффициент сжатия flex-элемента относительно других flex-элементов

Чем отличаются CSS Grid свойства justify-items, justify-content и justify-self?

justify-content позволяет вам расположить сетку внутри ее контейнера, justify-self позволяет переопределить положение контента по умолчанию в отдельной ячейке. Это заменит позицию, заданную justify-items. Это свойство содержимого внутри элемента сетки.

Зачем используется свойство grid-area?

grid-area это то свойство, которое принимает в качестве значения номера четырех грид-линий, определяющих расположение грид-области.

Сокращением от каких свойств является свойство grid-gap? Сколько значений они принимают?

войство grid-gap - это сокращение для grid-row-gap и grid-column-gap. Промежутки между ячейками (грид-интервалы) принимают одно значение для строк и одно значение для столбцов.
